요구사항 베이스라인

  1. 기능적 요구사항
    • 소프트웨어의 실질적 기능과 사용자의 요구사항과 연관이 되어있는 개념
    • 소프트웨어 구성에서 가장 중요한 요구사항임
      • 데이터 : 입력 데이터와 출력 데이터는 무엇이며, 데이터의 형식은 무엇인가?
      • 사용자 : 본 프로그램을 사용하는 사용자는 누구이며, 본 프로그램의 대상자가 명확한가?
      • 기능성 : 본 프로그램이 사용자에게 제공해야 하는 것은 무엇인가?
  2. 비기능적 요구사항
    • 문화 요구사항 : 한/영 지원, 날짜 양식 지원
    • 자원 요구사항 : 최소한의 메모리 지원, 다양한 하드웨어 지원
    • 성능 요구사항 : 응답시간은 1초 이내, DB 실시간 업데이트
    • 보안 요구사항 : 사용자별 접근권한 제한
    • 운영 요구사항 : OS 지원 다양화, 문서 양식에 제한받지 않아야 함
  3. 인터페이스 요구사항
    • 지원 인터페이스에 따른 요구사항 (표준)
